# Character Sprite Sheet

## Save Your Sprite Sheet Here

**File name**: `all_characters.png`

**Full path**: `assets/characters/all_characters.png`

## Specifications

- **Format**: PNG with transparency
- **Frame size**: 64×64 pixels
- **Layout**: 10 rows × 8 columns (80 frames total)
- **Total size**: 512×640 pixels (8×64 wide, 10×64 tall)

## Character Order (Top to Bottom)

1. Red Ninja (char1)
2. Blue Ninja (char2)
3. Green Soldier (char3)
4. Yellow Wizard (char4)
5. Purple Hero (char5)
6. Orange Pilot (char6)
7. Cyan Alien (char7)
8. Pink Knight (char8)
9. Brown Worker (char9)
10. Green Slime (char10)

## Frame Order (Left to Right)

Each character row has 8 frames:
1. IDLE
2. RUN-1
3. RUN-2
4. JUMP
5. FALL
6. LAND
7. HURT
8. DEAD

## How to Add Your Sprite Sheet

1. Save your sprite sheet image as `all_characters.png`
2. Place it in this folder (`assets/characters/`)
3. Refresh the game (Ctrl+Shift+R)
4. The game will automatically load your sprites!

## Fallback

If the file is not found, the game will use procedurally-generated sprites instead.
